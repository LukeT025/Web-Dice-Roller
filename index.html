Web Dice Roller

Rolls five 6-sided dice. Numbers are generated on the server. Auto-rolls on load.

Instructions

  • The page rolls automatically on load (after waking the server).
  • “Roll Dice” is focused by default. Press Enter to roll again.
  • All random numbers are fetched from the server via REST APIs.
  • Use “Test CORS Failure” to demonstrate a blocked cross-origin call.
